# AthleteTrack

AthleteTrack est une application SaaS de suivi athlète destinée aux professionnels du sport :

- kinésithérapeutes
- préparateurs physiques
- ostéopathes
- coachs sportifs

## Stack technique

Frontend :

- Vue 3
- Bootstrap 5
- Vue Router

Backend :

- FastAPI (à venir)

Cloud :

- AWS (à venir)

## Fonctionnalités prévues

- Authentification
- Dashboard athlètes
- Gestion séances
- Upload documents
- Résumés IA

## Installation

```bash
npm install
npm run dev
```
